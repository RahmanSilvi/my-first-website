Inline VS Block
An inline element does not start on a new line. An inline element only takes up as much width as necessary. For exsample: small tag, italic tag, emhasise tag, strong, bold etc.

On the other hand A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
Two commonly used block elements are: paragraph

and division

.
The paragraph

element defines a paragraph in an HTML document.
The division

element defines a division or a section in an HTML document.
